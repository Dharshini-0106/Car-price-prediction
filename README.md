## 📌 Overview
A machine learning regression model to predict Audi car prices 
based on key vehicle features. Multiple regression algorithms 
were compared to identify the best performing model.

## 📊 Dataset
- **Source:** Audi car price dataset
- **Features:** Model, year, transmission, mileage, fuel type, 
  engine size and more
- **Target:** Car price (in GBP)

## ⚙️ Methodology
- ✅ Exploratory Data Analysis
- ✅ Label Encoding and One Hot Encoding for categorical features
- ✅ Standard Scaling for feature normalization
- ✅ Train/Test split (80/20)
- ✅ Compared Random Forest, Linear Regression, Extra Trees Regressor
- ✅ Evaluated using R2 Score and Mean Absolute Error
